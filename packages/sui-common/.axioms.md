# AXIOMS

## AX-SUI-COMMON-001: CalendarBooking export contract is semver-bound
The CalendarBooking component exported from @stoked-ui/common must export
`CalendarBookingProps`, `BookingFormData`, `TimeSlot`, and `DayColumn` from its
types file. Any structural change to these exported types is a semver-breaking
change and requires a version bump before publishing.

### Applies To
- `packages/sui-common/src/CalendarBooking/CalendarBooking.types.ts`
- `packages/sui-common/src/CalendarBooking/index.tsx`

### Acceptance Checks
- `grep -n "TimeSlot\|DayColumn\|CalendarBookingProps\|BookingFormData" packages/sui-common/src/CalendarBooking/CalendarBooking.types.ts` shows all four exported
- `pnpm typescript` in packages/sui-common exits 0
