## Which download should I use?

- **Installer package (`.pkg`)** -- Recommended for most users. Installs the plugin into the standard OBS plugins location.
- **Zip archive (`.zip`)** -- Best if you prefer to inspect the bundle or install it manually.

## After Downloading

1. Install the package or extract the zip.
2. Open OBS Studio.
3. Add the **Auto-Focus Window Capture** source to a scene.
4. Grant Screen Recording and Accessibility permissions when prompted.
