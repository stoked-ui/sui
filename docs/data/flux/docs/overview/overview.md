# Overview

<p class="description">Flux turns any website into a living, interactive desktop wallpaper for macOS.</p>

## What is Flux?

Flux is a macOS application that renders any website -- remote or local -- as your desktop wallpaper. Whether it is an animated WebGL demo, a live data dashboard, a news feed, or the built-in fluid simulation, Flux places it behind your windows so your desktop is never static again.

## Key Features

- **Any Website as Wallpaper** -- Display remote URLs or local HTML files directly on your desktop.
- **Multi-Website Management** -- Add multiple websites, then switch between them freely.
- **Multi-Monitor Support** -- Assign a different website to each connected display.
- **Interactive Browsing Mode** -- Right-click to navigate, pinch-to-zoom, and interact with the page as if it were a browser tab.
- **Custom CSS & JavaScript Injection** -- Inject your own stylesheets and scripts into every loaded page.
- **Mouse Position Tracking** -- Websites receive live cursor coordinates, enabling interactive visual effects.
- **Color Inversion** -- Instant fake dark mode for sites that lack one.
- **Opacity Control** -- Adjust wallpaper transparency to keep the desktop readable.
- **Auto-Reload** -- Set an interval and Flux refreshes the page automatically.
- **URL Scheme Commands** -- Control Flux from scripts, Shortcuts, or other apps via `flux:` URLs.
- **Shortcuts & Siri Automation** -- First-class integration with the macOS Shortcuts app.
- **Built-in Fluid Simulation** -- A ready-to-use interactive fluid wallpaper ships with the app.

## Getting Started

1. **Install** -- [Download the Flux beta](/flux/docs/download) and follow the installation instructions.
2. **Add a Website** -- Open Flux from the menu bar and paste a URL, or use the built-in fluid simulation.
3. **Customize** -- Adjust opacity, inject CSS/JS, or enable color inversion from the settings panel.
4. **Automate** -- Use Shortcuts or URL schemes to switch wallpapers on a schedule or trigger.

Continue to the [Websites](/flux/docs/websites) page to learn how to add and manage wallpaper websites, or jump to [Scripting & Automation](/flux/docs/scripting) for URL schemes and the JavaScript API.
