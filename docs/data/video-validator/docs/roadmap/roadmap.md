# Roadmap

<p class="description">Planned features and improvements for the @stoked-ui/video-validator package.</p>

## Upcoming

- **Perceptual comparison** — SSIM and PSNR metrics alongside pixel matching
- **Audio validation** — Waveform comparison for audio tracks
- **Timeline markers** — Validate specific timestamp ranges instead of full video
- **GitHub Actions integration** — Pre-built action for CI/CD pipelines
- **Visual diff viewer** — Interactive HTML report with side-by-side frame comparison and diff overlay

## Recently completed

- Frame-by-frame pixel comparison
- Batch validation support
- HTML and JSON report generation
- CLI with validate, batch, and extract commands
- Configurable thresholds and frame intervals
