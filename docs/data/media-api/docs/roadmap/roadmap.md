# Roadmap

<p class="description">Planned features and improvements for the @stoked-ui/media-api package.</p>

## Upcoming

- **CDN integration** — CloudFront distribution for optimized media delivery
- **Transcoding pipeline** — Automatic video transcoding to multiple resolutions and formats
- **Webhook events** — Notify external services on upload completion, processing, and deletion
- **Rate limiting** — Configurable rate limits per user and endpoint
- **Analytics** — View counts, bandwidth usage, and storage metrics
- **Multi-tenant support** — Organization-level media isolation

## Recently completed

- CRUD operations for media items
- Resumable multipart upload flow
- Metadata extraction (FFmpeg + Sharp)
- Thumbnail generation
- Search and filtering
- JWT authentication
- Swagger/OpenAPI documentation
