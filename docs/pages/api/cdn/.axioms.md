# CDN API Axioms

- The public CDN directory API contract is path-based: `https://cdn.stokd.cloud/<directory>/?f=json` or `?f=yaml`.
- Do not require public callers to construct a `prefix=` query string for directory listings.
- Internal handlers may still accept `prefix=` for admin tooling and app-internal fetches, but that is not the public integration surface.
- Edge rewrites for CDN directory listings must not depend on forwarding synthetic query strings from non-`/api/*` cache behaviors.
