Stoked UI Documentation

Comprehensive UI component library for React. Includes Media, Timeline, File Explorer, Editor, and Flux components.

Getting Started

Install the packages you need:

npm install @stoked-ui/media
npm install @stoked-ui/timeline
npm install @stoked-ui/file-explorer
npm install @stoked-ui/editor

Products

  • Media — MediaViewer, MediaCard, MediaFile components
  • Timeline — Visual timeline editor with tracks and keyframes
  • File Explorer — Tree-based file browser component
  • Editor — Rich media editing suite
  • Flux — AI-powered workflow automation
