Editor

A rich media editing suite combining the Timeline, File Explorer, and Media components into a professional non-linear video editor.

Installation

npm install @stoked-ui/editor

Overview

The Editor package provides a complete non-linear video editing experience in the browser. It combines the Timeline for sequencing, the File Explorer for asset management, and Media components for playback into a unified editing interface.

Architecture

The editor is composed of three main panels:

  • Asset Panel — File Explorer for managing media assets (powered by @stoked-ui/file-explorer)
  • Preview Panel — MediaViewer for real-time preview of the current timeline composition
  • Timeline Panel — Multi-track timeline for sequencing clips (powered by @stoked-ui/timeline)

Key Features

  • Non-linear video editing with multi-track support
  • Real-time preview during timeline scrubbing
  • Asset management with drag-and-drop onto timeline
  • Export to video using WebCodecs API or WASM renderer
  • Project save/load with JSON serialization
  • Keyboard shortcuts for all common operations

Basic Usage

import { Editor } from '@stoked-ui/editor';

function VideoEditorApp() {
  return (
    <Editor
      project={myProject}
      onExport={(output) => downloadFile(output)}
    />
  );
}
