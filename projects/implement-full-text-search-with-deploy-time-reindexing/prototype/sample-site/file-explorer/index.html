File Explorer

A tree-based file browser component for React with drag-and-drop, multi-select, and virtual file system support.

Installation

npm install @stoked-ui/file-explorer

Overview

The File Explorer component renders a hierarchical file tree with support for expanding and collapsing directories, drag-and-drop reordering, multi-select with keyboard modifiers, and custom icons per file type. Built on top of MUI X Tree View.

Key Features

  • Hierarchical tree rendering with lazy loading
  • Drag-and-drop file and folder reordering
  • Multi-select with Shift+Click and Cmd+Click
  • Custom icons per file type (image, video, audio, document)
  • Context menu with cut, copy, paste, rename, delete actions
  • Keyboard navigation (arrow keys, Enter to open, Delete to remove)
  • Virtual file system adapter for non-native environments

Basic Usage

import { FileExplorer } from '@stoked-ui/file-explorer';

const files = [
  { id: '1', name: 'Documents', type: 'folder', children: [
    { id: '2', name: 'report.pdf', type: 'document' },
    { id: '3', name: 'photo.jpg', type: 'image' },
  ]},
  { id: '4', name: 'video.mp4', type: 'video' },
];

function App() {
  return (
    <FileExplorer
      items={files}
      onNodeSelect={(node) => console.log('Selected:', node)}
    />
  );
}
