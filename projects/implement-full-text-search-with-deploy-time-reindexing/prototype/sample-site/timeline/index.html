Timeline

A visual timeline editor component for React with support for tracks, keyframes, and media playback.

Installation

npm install @stoked-ui/timeline

Overview

The Timeline component provides a professional video-editor-style timeline interface. Drag tracks, add keyframes, and synchronize media playback. Built for complex animation and video editing workflows.

Key Features

  • Multi-track timeline with drag-and-drop support
  • Keyframe editing with bezier curve interpolation
  • Frame-accurate playback scrubbing
  • Zoom and pan controls
  • Track locking and visibility toggles
  • Undo/redo history stack

Basic Usage

import { Timeline, TimelineTrack } from '@stoked-ui/timeline';

function VideoEditor() {
  return (
    <Timeline duration={60} fps={30}>
      <TimelineTrack name="Video Layer 1" src="/assets/clip1.mp4" />
      <TimelineTrack name="Audio Track" src="/assets/audio.mp3" />
    </Timeline>
  );
}

Props

PropTypeDefaultDescription
durationnumber30Total timeline duration in seconds
fpsnumber24Frames per second
currentTimenumber0Current playhead position
onTimeChange(time: number) => void—Callback when playhead moves
zoomnumber1Zoom level (1 = fit to container)
