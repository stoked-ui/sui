# Infra Axioms

- `cdn.stokedconsulting.com/<directory>/?f=json|yaml` is a stable public directory-listing API and must resolve to that directory, not bucket root.
- CDN edge logic must preserve the path-based API contract without requiring callers to know about internal `prefix=` query parameters.
- Do not rewrite public CDN listing requests in a way that depends on non-API cache behaviors forwarding injected query strings.
